# Earth/Co Market

Earth/Co Market is a full-stack ecommerce demo with an earth-brutalist visual style. It includes a storefront, cart and checkout flow, auth, and admin product management.

## Highlights

- React storefront with product, cart, account, and checkout flows
- PostgreSQL and Prisma-backed backend
- Stripe test checkout integration
- Admin CRUD for product management

## Stack

- React + Vite
- TypeScript
- Prisma + PostgreSQL
- Stripe

## Quick start

```bash
bun install
bun run db:up
bun run dev
```

Useful commands:

```bash
bun run dev:server
bun run db:migrate
bun test
```
